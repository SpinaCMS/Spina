# Rendering content

Spina features 7 built-in parts that you can use to build pages.

- `Spina::Parts::Line`
- `Spina::Parts::Text`
- `Spina::Parts::Image`
- `Spina::Parts::ImageCollection`
- `Spina::Parts::Attachment`
- `Spina::Parts::Option`
- `Spina::Parts::Repeater`

Learn more about them in the next chapters.
